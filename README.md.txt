# C++ KeyBinder

Простой и легковесный кроссплатформенный биндер для Windows, который работает в фоновом режиме и позволяет открывать программы, сайты или файлы по нажатию комбинации клавиш.

## Особенности
- **Фоновый режим**: после настройки окно консоли скрывается, программа не мешает работе.
- **Множественные бинды**: можно настроить неограниченное количество комбинаций.
- **Сохранение настроек**: все бинды автоматически сохраняются в файл `config.txt` рядом с программой.
- **0% нагрузки на ЦП**: оптимизированный цикл ожидания клавиш.

## Как использовать
1. Запустите скомпилированный файл.
2. Следуйте инструкциям на экране, чтобы добавить новую комбинацию (введите путь/ссылку и нажмите две клавиши).
3. Программа скроется в фон. Чтобы вызвать бинд, зажмите выбранные клавиши одновременно.

## Как закрыть программу
Так как программа работает в скрытом фоновом режиме, её можно закрыть через **Диспетчер задач** Windows (найдите процесс с именем вашей программы и нажмите «Снять задачу»).
